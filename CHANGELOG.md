# Changelog

## 0.3.0

- Added adventuring party management.
- Added adventuring party editing.
- Added encounter setup for selected adventuring party characters.
- Added allies and opponents to encounter setup.
- Added encounter editing.
- Added run encounter mode.
- Added editable encounter participant hit points with max hit point enforcement.
- Added incapacitated tracking for encounter participants.
- Added drag-and-drop ordering for encounter participants.
- Added initiative tracking with highest-to-lowest ordering after leaving the initiative field.
- Added Dnd Adventure Club as a selectable game system.

## 0.2.0

- Added character management.
- Added character persistence.
- Added D&D 5e as the first supported game system.
- Added character hit point fields with current hit points initialized from max hit points.

## 0.1.0

- Added player management.
- Added player rename support.
- Added local SQLite persistence.
- Added the initial Electron, React, and TypeScript app foundation.
