# Changelog

## 0.2.0

- Added character management.
- Added character persistence.
- Added D&D 5e as the first supported game system.
- Added character hit point fields with current hit points initialized from max hit points.

## 0.1.0

- Added player management.
- Added player rename support.
- Added local SQLite persistence.
- Added the initial Electron, React, and TypeScript app foundation.
